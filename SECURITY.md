# Security Policy

## Supported Versions

Chromarchy is an actively maintained Unity Editor tool. Security updates are provided for the latest release on the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in Chromarchy, please report it privately rather than opening a public issue.

The preferred way to report is through GitHub's private vulnerability reporting:

1. Go to the **Security** tab of this repository.
2. Click **Report a vulnerability**.
3. Provide as much detail as possible, including steps to reproduce.

We will acknowledge your report as soon as possible and keep you informed about the progress toward a fix.

## Scope

This tool runs only inside the Unity Editor and has no runtime footprint in builds. Please keep this in mind when assessing potential impact.

Thank you for helping keep Chromarchy and its users safe.
